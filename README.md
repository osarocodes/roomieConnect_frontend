# Roomie Connect

## Local Setup
First clone the repo locally using your prefered method. E.g
```bash
git clone https://github.com/osarocodes/RoomieConnect
```
Enter the directory of the cloned repo:
```bash
cd RoomieConnect
```
Install npm packages:
```bash
npm i # or npm install
```
Build Project:
```bash
npm run build
```
Start local server:
```bash
npm run dev
```

## About Roomie Connect

Roomie Connect is a web application designed to help students and young professionals find compatible roommates and manage shared living arrangements. The platform allows users to create profiles, upload verification documents, specify preferences, and match with potential roommates based on personality, lifestyle, and housing needs.

### Key Features

- **Profile Creation:** Users can set up personal profiles with photos and details.
- **Document Upload:** Securely upload admission letters or other verification documents.
- **Roommate Matching:** Match with others based on preferences and personality traits.
- **Progress Tracking:** Visual progress bar guides users through the signup process.
- **Admin Dashboard:** Manage users and view analytics (for administrators).

Roomie Connect is built with React, Vite, and Tailwind CSS for a fast, modern, and responsive experience.
